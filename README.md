# Maternal-Health-Risk-Analysis

## Project Overview
This project uses machine learning to predict maternal health risk levels (Low, Mid, and High) based on maternal health indicators. The project includes data preprocessing, exploratory data analysis, model development, and an interactive dashboard.

## Objectives
- Predict maternal health risk levels.
- Identify the most important risk factors.
- Support early intervention through data-driven insights.

## Dataset
The project contains:
- Original maternal health dataset
- Cleaned dataset used for machine learning

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Zerve
- GitHub

## Repository Structure

data/        # Raw and cleaned datasets
images/      # Dashboard screenshots
models/      # Trained machine learning model
notebooks/   # Jupyter notebooks
reports/     # Final project report

## Machine Learning Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Risk Prediction Dashboard

## Results
The model predicts maternal health risk levels and provides visual insights through an interactive dashboard to support healthcare decision-making.
